jQuery.Open.Soccer.StatsSoccer Multileague Stats jQuery Plugin

This plugin shows soccer stats in your webpages.
It can show standings about italian "Serie A" and "Serie B", british "Premiere League" and spanish "Liga"

Include jQuery and gOpenSoccer

Include javascript files in your page. Remember that jQuery is reuqired!

<script src="jquery.min.js"></script>
<script src="jquery.opensoccer.min.js"></script>

Plugin Usage

Use plugin in this way. All parameters used on example below, are setted as default value. 

$('#yourElementID').JOS_getStats({
    season: "15-16",
    league: "serie-a",
    debug: false
});
Option Type Default Description and Allowed Values
season STRING "15-16" Defines the standings season
Allowed:
"15-16", "16-17"
league STRING "serie-a" Defines the standings league
Allowed:
"serie-a", "serie-b", "premiere-league", "liga"
team STRING/BOOL false Defines the standings data that will be showed by plugin.
Allowed:
"all", "Single Team Name", false
apiBaseURL STRING "http://soccer.sportsopendata.net/v1/leagues/" Defines the base Sports Open Data API URL.
Don't change this value!
Allowed:
"any valid url"
