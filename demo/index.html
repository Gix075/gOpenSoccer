g.Open.Soccer.STandingsSoccer Multileague Standings jQuery Plugin

This plugin shows soccer standings in your webpages.
It can show standings about italian "Serie A" and "Serie B", british "Premiere League" and spanish "Liga"

Include jQuery and gOpenSoccer

Include javascript files in your page. Remember that jQuery is reuqired!

<script src="jquery.min.js"></script>
<script src="jquery.gopensoccer.min.js"></script>

Plugin Usage

Use plugin in this way. All parameters used on example below, are setted as default value. 

$('#yourElementID').gOpenSoccer_getStandings({
    season: "15-16",
    league: "serie-a",
    view: "extended",
    view: "simple",
    template: "table",
    debug: false
});
Option Type Default Description and Allowed Values
season STRING "15-16" Defines the standings season
Allowed:
"15-16", "16-17"
league STRING "serie-a" Defines the standings league
Allowed:
"serie-a", "serie-b", "premiere-league", "liga"
view STRING "simple" Defines the standings data that will be showed by plugin.
Allowed:
"simple", "advanced", "extended", "full"
template STRING "table" Defines the standings output template.
Allowed:
"table", "tableless"
apiBaseURL STRING "http://soccer.sportsopendata.net/v1/leagues/" Defines the base Sports Open Data API URL.
Don't change this value!
Allowed:
"any valid url"

Table

This is a template with html table markup. It returns out the code that you must place inside tag <tbody>. In this way you will free to use your own labels on table head.
To use table template with simple view, you must place a table in your code like the example above: 

<table>
    <thead>
        <tr>
            <th>Position Label</th>
            <th>Team Name</th>
            <th>Points</th>
            <th>Matches Played</th>
        </tr>
    </thead>
    <tbody id="standings">
    </tbody>
</table>

<script>
    $('#standings').gOpenSoccer_getStandings({
        season: "15-16",
        league: "serie-a",
        view: "simple",
        template: "table"
    });
</script>
